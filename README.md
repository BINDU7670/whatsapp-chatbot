WhatsApp Chatbot using Flask & Twilio

📌 Project Description
This project is a WhatsApp chatbot built using Python, Flask, and Twilio API.  
It can respond to user queries like courses, fees, and placements using a simple NLP model.

---

🚀 Features
- WhatsApp integration using Twilio Sandbox
- NLP-based chatbot using Machine Learning
- Handles multiple user queries
- Smart responses for AI, courses, fees, etc.
- Stores user messages in CSV file
- Real-time communication using Flask

---

🛠 Technologies Used
- Python
- Flask
- Twilio API
- Scikit-learn
- Ngrok
